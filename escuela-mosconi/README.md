# Sitio web de la Escuela Mosconi

Sitio web institucional desarrollado para la Escuela 4-022 Enrique Mosconi, ubicada en Chacras de Coria, Mendoza.

## Objetivo

Mejorar la presencia digital de la institución, facilitar la comunicación con las familias y presentar su oferta educativa, instalaciones y actividades.

## Secciones principales

- Inicio.
- Información institucional.
- Historia, misión y visión.
- Espacios curriculares.
- Perfiles de egreso.
- Instalaciones.
- Blog de noticias y eventos.
- Información de contacto.

## Características

- Diseño adaptable a computadoras y dispositivos móviles.
- Menú de navegación responsivo.
- Carruseles de imágenes.
- Animaciones durante el desplazamiento.
- Contadores informativos.
- Organización mediante componentes reutilizables de PHP.

## Tecnologías utilizadas

- PHP
- HTML
- CSS
- JavaScript

## Estado del proyecto

Proyecto en desarrollo. La estructura principal y las secciones institucionales se encuentran implementadas. El blog y las funcionalidades dinámicas están planificados para una etapa posterior.

## Privacidad

No se incluyen datos personales de estudiantes, credenciales ni documentación interna de la institución.
