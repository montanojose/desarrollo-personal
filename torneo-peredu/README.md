# Torneo Personal Educativo

Plataforma web desarrollada para administrar la Copa PER.EDU, un torneo de fútbol destinado al personal educativo.

## Funcionalidades principales

- Registro e inicio de sesión de usuarios.
- Gestión de jugadores, capitanes y administradores.
- Inscripción de jugadores en equipos.
- Administración de torneos, fechas y partidos.
- Registro de asistencia.
- Carga de goles, tarjetas y resultados.
- Generación de tablas de posiciones.
- Gestión de fases eliminatorias.

## Tecnologías utilizadas

- PHP
- MySQL
- HTML
- CSS
- JavaScript

## Estado del proyecto

Proyecto funcional y publicado en un servidor web.

La documentación y los archivos del sistema se incorporarán progresivamente.
