# Piccolo Cortinas

Sitio web comercial desarrollado para una empresa dedicada a la venta, colocación y reparación de cortinas en Mendoza.

## Objetivo

Presentar los productos y servicios de la empresa, facilitar el contacto con potenciales clientes y mejorar su presencia digital.

## Funcionalidades principales

- Página de inicio con presentación comercial.
- Carrusel de imágenes.
- Catálogo de productos destacados.
- Información sobre servicios y beneficios.
- Sección institucional.
- Formulario de contacto.
- Integración con WhatsApp.
- Diseño adaptable a computadoras y dispositivos móviles.
- Animaciones y efectos de desplazamiento.

## Tecnologías utilizadas

- PHP
- HTML
- CSS
- JavaScript

## Estado del proyecto

Frontend desarrollado y funcional. La incorporación del backend y del panel de administración está planificada para una etapa posterior.

## Privacidad

Los datos personales, credenciales y configuraciones privadas del cliente no se incluyen en este repositorio.
